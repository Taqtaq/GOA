What is HTML?

HTML არის ენა, რომლითაც ვქმნით ვებ გვერდის სტრუქტურას. HTML-ში ვიყენებთ ტეგებს როგორიცაა: h1, p, a, img, ul და სხვ.

  • HTML = Structure
  • Tag-ები აღწერს ელემენტებს
  • Browser კითხულობს HTML-ს და აჩვენებს გვერდს
