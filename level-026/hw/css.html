What is CSS?

CSS არის ენა, რომელიც HTML ელემენტებს აძლევს სტილს: ფერს, ზომას, ფონს, დაშორებებს და განლაგებას.

  1. Selector - რას ვსტილავთ (მაგ: p, .class, #id)
  2. Property - რა მახასიათებელია (color, font-size...)
  3. Value - მნიშვნელობა (red, 20px...)
